# OTGW Logger, MQTT & Telegram Monitor

Легкий и надежный Docker-контейнер для логирования работы OpenTherm Gateway.

## Возможности
**Логи:** Сохраняет историю работы котла в файлы (с ротацией по часам и дням).
**MQTT:** Интеграция с Home Assistant — отправляет статус ошибок (`Error 01` и др.).
**Telegram:** Мгновенные уведомления об авариях и статусе подключения прямо на телефон.
**Фильтр:** Очищает логи от системного мусора и цветовых кодов.

## Установка (Быстрый старт)

1. **Скачайте файлы** репозитория в папку на сервере.

2. **Создайте файл настроек:**
   Вам нужно переименовать файл-шаблон `.env.example` в `.env`:
   ```bash
   cp .env.example .env
3.   Заполните настройки: Откройте файл .env и впишите свои данные (IP шлюза, пароли MQTT, токен Telegram): 
4.   Запустите контейнер:
   Логи сохраняются в папку ./logs рядом с файлом запуска.
