# OTGW Smart Logger & Monitor v3.0

Продвинутый Docker-контейнер для OpenTherm Gateway. Превращает "глупой" поток данных в умный мониторинг.

##  Возможности v3.0
* **Декодер OpenTherm:** Скрипт читает HEX-коды и понимает реальные значения (Температура, Давление, Модуляция).
* **Telegram Бот:**
  *  Мгновенные алерты при ошибках (с расшифровкой кодов на русском).
  *  Регулярные отчеты раз в 6 часов (Температура, Давление, Статус ошибок).
* **MQTT:** Передача критических ошибок в Home Assistant.
* **Логирование:** Чистые логи с ротацией.

## Установка
1. **Скачайте код.**
2. **Настройте:** Переименуйте `.env.example` в `.env` и впишите IP шлюза, MQTT и Telegram токены.
3. **Запустите:**
   ```bash
   docker compose up -d
