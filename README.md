# OTGW Logger & MQTT Monitor

Легкий Docker-контейнер для логирования работы OpenTherm Gateway.
* **Логи:** Пишет чистые данные в файлы (ротация по часам/дням).
* **Мониторинг:** Шлет уведомления об ошибках (`Error 01`) в Home Assistant через MQTT.
* **Фильтр:** Убирает системный мусор и ANSI-цвета из логов.

## Быстрый старт

1. Скачайте файлы.
2. Создайте файл настроек из шаблона:
   ```bash
   cp .env.example .env
